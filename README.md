# webservice-notify-ws
Um serviço da web para notificação


## Start
Use o código para instalar as dependencias:
````shell
npm install
````
Em seguida, inicie o servidor executando:
````shell
npm start
````

## Tecnologias
Express, Socket.io, Axios
